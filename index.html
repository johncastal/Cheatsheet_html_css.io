  1. Inline
  2. ID (#id)

  4. Elements and pseudo-elements

Selector's Types

Inline

This selector es defined within HTML code. To do that is needed to use key: "style:"

ID

This is in CSS the most specific selector. In CSS file is called as "#ID:"

Classes

With this selector is possible to define classes within HTML elements. CSS is defined as ".class:".

Attributes

Allow define propertis through the keyword "target" in HTML. This is used to build specifically within the same HTML elements. CSS is defined as "element[target]"

Pseudo classes

It is a keyword that is added to selectors and specifies a special state of a selected element. Is defined as "selector:pseudoclass".

Elements

Correspond to elements defined in HTML such as "h","p","ul","li".

Pseudo elements

Is used to modify parts of elements in HTML or add a special style. In CSS is defined as "element::first-line".
